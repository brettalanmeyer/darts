Darts API

GET /api/players/

Array of player data.

[{
	id: 1,
	name: "Morgan Freeman",
	createdAt: '2016-01-01 12:01:01'
}]
GET /api/matches/

Array of match data Includes information about progress of rounds and player turns.

[{
	id: 1,
	modeId: 2,
	players: 4,
	game: 1,
	round: 13
	ready: true,
	turn: 3,
	complete: false
	createdAt: '2016-01-01 12:01:01'
}]
GET /api/marks/

Player marks per round and what value was hit.

[{
	id: 1,
	matchId: 1,
	teamId: 3,
	playerId: 7,
	game: 9,
	round: 3,
	value: 20,
	createdAt: '2016-01-01 12:01:01'
}]
GET /api/teams/

Teams and their win/loss after a complete game.

[{
	id: 1,
	matchId: 1,
	win: true,
	loss: false
}]
GET /api/teams-players/

Relationship between teams and players

[{
	id: 1,
	teamId: 21,
	playerId: 34
}]
