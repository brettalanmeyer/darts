## Config
Create a config file here: ```/darts/config_file.cfg``` and specify the following parameters:
```
DEBUG=True
PORT=5000
MYSQL_USERNAME=''
MYSQL_PASSWORD=''
MYSQL_HOST=''
MYSQL_DATABASE=''
```
